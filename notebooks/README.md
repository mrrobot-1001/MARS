# Colab Notebook Templates

These files document the required notebook flow for Phase 1:

1. Mount cloud storage.
2. Install dependencies.
3. Load processed data.
4. Split train/validation/test by segment to reduce leakage.
5. Train and evaluate.
6. Export artifact and deployment card.

Use the Python modules in `src/mars_ml_pipeline/training/` from Colab to keep local and cloud training behavior aligned.
